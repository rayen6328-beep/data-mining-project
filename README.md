steam Data Mining Project

Project Overview :
This project analyses a Steam games dataset using Python and data mining techniques.

The project includes:

Data loading
Data preprocessing
Missing value analysis
Descriptive statistics
Exploratory data analysis
Data visualisation
Correlation analysis
Rating ratio calculation

Technologies Used :

Python
Pandas
Matplotlib
Seaborn
Jupyter Notebook

Dataset

Steam Store Games Dataset (Kaggle) :
